Introduction

MAES (Moderation and Enforcement System) is an admin framework built for Roblox developers who want full control over their moderation tools.

Installation

  1. Download MAES from the official Discord.
  2. Place the Server folder into ServerScriptService.
  3. Edit your BloxOLite module to match your group settings.

Configuration

return {
  ["General_Settings"] = {
    ["prefix"] = ";",
    ["Hotkey"] = "h",
    ["BanData"] = data:GetDataStore("BanData123"),
    ["WebHookUrl"] = "https://discord.com/api/webhooks/...",
  },
}
