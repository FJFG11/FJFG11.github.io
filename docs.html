Set-up

Everything you need to know about setting up BloxOLite products.

Introduction

BloxOLite Anti-Cheat is a next-gen anti-cheat that fights against exploiters with constant updates..

Installation

Follow these steps to set up the Anti-Cheat in your game:

  1. Download AC from the official Discord.
  2. Place the Anti-Cheat module into ServerScriptService.
  3. To edit the settings go-to ServerScriptService -> Ant-Cheat

General Configuration

["Options"] = {
		["AttributeName"] = "StrikesLogs", -- Name of attribute to store strikes
		["Ammount"] = 3, -- Amount of strikes before punishment
		["Punishment_Webhook_Url"] = "https://discord.com/api/webhooks/", -- Put your wbehook url here

		["BanDataStore"] = data:GetDataStore("BanData4"), -- Your ban DataStore

		["Punishment"] = "kick", -- Punishment
		--[[
			[Kick] = Kicks user
			[ban] = bans user
		]]

		["BanMessage"] = "AC : Banned for triggering Anti-cheat.", -- Ban Message
		["BannedMessage"] = "You are banned from this game due to anti-cheat violations.", -- Banned Message
		["KickMessage"] = "AC : Kicked for triggering Anti-cheat.", -- kick Message

		["WhitelistUsers"] = {  
--			[123456789] = true, -- Example: user 123456789 can bypass anti-cheat
		},

		["WhitelistGroups"] = {
			[591239665] = 3,   -- Example: must have rank 150+ in group 123456789 to bypass anti-cheat
		},

	},

	["Detections"] = {
		["Studs"] = 3, -- studs moved in one frame
		["Speed-Threshold"] = 20, -- Speed threshold
		["Speed-Margin"] = 4, -- Speed_Threshold + margin = max speed to help stop false positives
		["Invis-Threshold"] = 0.5, -- Invis Threshold
		["NoClip-Threshold"] = 1, -- NoClip Threshold
		["Lag-Threshold"] = 2, -- Lag Threshold (Keep above 2 to stop false positives)
	},
}
}

Types of punishments

When editing the Anti-Cheat settings you will come across ["Punishment"] = "???" and there is 2 types of punishments Kick and Ban.
The first type is kick and it just kicks the user from the server they can rejoin.
The second type is ban and it permbans the user from you game and can only be unbaned by reseting the BanDataStore.

Reseting the BanDataStore will unban all bans.
