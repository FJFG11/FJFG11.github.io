Set-up

Everything you need to know about setting up BloxOLite products.

Introduction

BloxOLite Anti-Cheat is a next-gen anti-cheat that fights against exploiters with constant updates.

Installation

Follow these steps:

  1. Download AC from the official Discord.
  2. Place Anti-Cheat in ServerScriptService.
  3. To edit settings for the anti-cheat go-to ServerScriptService -> Anti-Cheat.

General Configuration

[["Options"] = {
		["AttributeName"] = "StrikesLogs", -- Name of attribute to store strikes
		["Ammount"] = 3, -- Amount of strikes before punishment
		["Punishment_Webhook_Url"] = "https://discord.com/api/webhooks/", -- Put your wbehook url here

		["BanDataStore"] = data:GetDataStore("BanData4"), -- Your ban DataStore

		["Punishment"] = "kick", -- Punishment
		--[[
			[Kick] = Kicks user
			[ban] = bans user
		]]

		["BanMessage"] = "AC : Banned for triggering Anti-cheat.", -- Ban Message
		["BannedMessage"] = "You are banned from this game due to anti-cheat violations.", -- Banned Message
		["KickMessage"] = "AC : Kicked for triggering Anti-cheat.", -- kick Message

		["WhitelistUsers"] = {  
--			[123456789] = true, -- Example: user 123456789 can bypass anti-cheat
		},

		["WhitelistGroups"] = {
			[591239665] = 3,   -- Example: must have rank 150+ in group 123456789 to bypass anti-cheat
		},

	},

	["Detections"] = {
		["Studs"] = 3, -- studs moved in one frame
		["Speed-Threshold"] = 20, -- Speed threshold
		["Speed-Margin"] = 4, -- Speed_Threshold + margin = max speed to help stop false positives
		["Invis-Threshold"] = 0.5, -- Invis Threshold
		["NoClip-Threshold"] = 1, -- NoClip Threshold
		["Lag-Threshold"] = 2, -- Lag Threshold (Keep above 2 to stop false positives)
	},

Types of punishments

When you are editing settings in the Anti-Cheat Module you will see ["Punishment"] = "???" and you have 2 choice:
Kick : Kicks the user who has tripped the anti-cheat. (They can rejoin)
Ban : Bans user permantly. (Cannot rejoin)

Introduction

BloxOLite Mod call allows players to request staff through a Gui.

Installation

Follow these steps:

  1. Download ModCall from the official Discord.
  2. Place ModCall folder in ServerScriptService.
  3. Place ModUI ScreenGui in StarterGui.
  4. To edit settings for the modcall go-to ServerScriptService -> ModCall -> Settings.

General Configuration

 -- General
Config.Enabled = true -- Is modcall enabled?
Config.IsGroupLocked = false -- Is the Gui group locked?
Config.CoolDown = 160 -- Cooldown

-- Group
Config.GroupID = 123456789 -- GroupID
Config.GroupRank = 50 -- Rank required

-- Webhook
Config.WebHookUrl = "https://discord.com/api/webhooks/..." -- Your webhook url

-- Webhook Meta
Config.Title = "Moderator Needed"
Config.Desc = "A moderator is required in the server below."
Config.SubTitle1 = "Reporter"
Config.SubTitle2 = "Reported Player"
Config.SubTitle3 = "Reason"
Config.SubTitle4 = "Server Information"
