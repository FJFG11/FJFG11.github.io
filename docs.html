Introduction

BloxOLite is a lightweight moderation and anti-exploit framework designed for Roblox developers. It integrates directly into your game and provides real-time tools for managing player activity, security, and behavior.

Installation

To install BloxOLite, follow these simple steps:

  1. Insert the folder into ServerScriptService.
  2. Configure your permissions and settings in the Settings module.
-- Example:
local BloxOLite = require(game.ServerScriptService.BloxOLite)
BloxOLite.Init()

Configuration

BloxOLite uses a single Settings module to define ranks, permissions, and command categories.

return {
  BaseRolePermissions = {
    ["Staff"] = { "kick", "ban", "warn" },
    ["Developer"] = { "kick", "ban", "unban", "logs" },
    ["Owner"] = { "kick", "ban", "unban", "logs", "announce" },
  },
  TBDS = "TempBanStore",
  MCPH = "ModerationLogs"
}
