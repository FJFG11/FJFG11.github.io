BloxOLite provides a powerful moderation framework for Roblox developers. With just a few steps, you can integrate advanced moderation commands, logging, and permission control into your experience.

‚öôÔ∏è Installation

Setting up BloxOLite is quick and easy:

  1. Download the latest release from GitHub Releases.
  2. Insert the module into ServerScriptService in your Roblox Studio project.
  3. Edit your Settings ModuleScript to configure permissions, group IDs, and DataStores.
