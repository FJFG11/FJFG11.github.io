Documentation

Everything you need to know about setting up and customizing MAES (Moderation and Enforcement System) for your Roblox experience.

Introduction

MAES is a open source moderation system also known as a Custom Admin system that can be customized to your liking.

Installation

To install MAES, follow these simple steps:

  1. Download MAES from the discord.
  2. Insert the Server into ServerScriptService.
  3. Configure your permissions and settings in the BloxOLite module.

General Settings

return {

    ["General_Settings"] = {
	  ["prefix"] = ";", -- Default prefix for commands
	  ["Hotkey"] = "h", -- Default hotkey to open the panel (IF THE SYSTEM BREAKS IT AUTO SETS TO F4)
	  ["BanData"] = data:GetDataStore("BanData123"), -- Datastore for ban data
	  ["Modlogs"] = data:GetDataStore("ModerationLogs123"), -- Datastore for moderation logs E.g ModerationLogs4
	  ["WebHookUrl"] = "https://discord.com/api/webhooks/...", -- Webhook Url for your ban commands (ban, serverban, idban, pban)
		
      ["BanPlaceEnabled"] = true, -- Does that game tp banned players when banned?
	  ["BanPlaceID"] = "1234567890" -- Ban Place ID
	},

}

Rank Settings

return {

     ["Guest"] = {
	   ["Rank_Needed"] = 0, -- Rank needed to use this rank
	   ["Enabled"] = true, -- If disabled this rank will not be able to use the panel or commands
	   ["Execute_Code"] = false, -- If enabled this rank will be able to execute code from the execute command
	   ["Usable_Commands"] = { -- Commands this rank can use (Ranks Above can also use these)
					
		   "panel", "cmds", "Clear", "help", "credits"
		  },
		},
        
}

Roblox Security Settings

These need to be on in your game for MAES to work.


        Allow HTTPS request = true (Remote Events)
        API Services = true (DataStores)
