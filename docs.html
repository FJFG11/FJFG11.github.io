🚀 Getting Started

BloxOLite provides easy-to-use moderation tools for your Roblox games. You can manage bans, kicks, logs, and permissions in one unified system.

⚙️ Installation

  1. Download the BloxOLite module from the GitHub Releases page.
  2. Insert it into your Roblox game’s ServerScriptService.
  3. Configure your settings inside the Settings ModuleScript.

🧩 Command Reference

Use the console or UI to run moderation commands.

  • !kick [player] [reason] — Kicks a player
  • !ban [player] [time] [reason] — Bans a player temporarily
  • !debughistory [player] — Views punishment logs

💡 API Integration

BloxOLite supports simple API endpoints for logging and audit purposes. Example:

{
  "event": "PlayerBanned",
  "player": "User123",
  "moderator": "Admin456",
  "reason": "Exploiting"
}
