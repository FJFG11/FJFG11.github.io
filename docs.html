MAES — Moderation & Enforcement

Introduction

MAES (Moderation and Enforcement System) is a admin framework built for Roblox developers who want full control over their moderation features.

Installation

  1. Download MAES from the official Discord.
  2. Place the Server folder into ServerScriptService.
  3. Edit your BloxOLite module.

General Configuration

return {
  ["General_Settings"] = {
    ["prefix"] = ";",
    ["Hotkey"] = "h",
    ["BanData"] = data:GetDataStore("BanData123"),
    ["Modlogs"] = data:GetDataStore("ModerationLogs123"),
    ["WebHookUrl"] = "https://discord.com/api/webhooks/...",

    ["BanPlaceEnabled"] = true,
    ["BanPlaceID"] = "1234567890",
  },
}

Rank & Permission Settings

return {
  ["Guest"] = {
    ["Rank_Needed"] = 0,
    ["Enabled"] = true,
    ["Execute_Code"] = false,
    ["Usable_Commands"] = {
      "panel","cmds","clear","help","credits"
    },
  },
}

Security

Allow HTTPS Requests = true
Enable API Services = true

Anti-Cheat Documentation

Introduction

BloxOLite Anti-Cheat protects your game from exploits.

Installation

  1. Download the Anti-Cheat.
  2. Insert file into your game.
  3. Drag Anti-Cheat V2 into ServerScriptService.

Configuration

Open ServerScriptService → Anti-Cheat → settings module.

ModCall Documentation

Introduction

BloxOLite ModCall allows players to call staff for help.

Installation

  1. Download from Discord.
  2. Insert into your game.
  3. Drag ModCall into ServerScriptService.
  4. Drag ModUI into StarterGui.
  5. Set your webhook URL.

Configuration

Open ServerScriptService → ModCall → settings.
