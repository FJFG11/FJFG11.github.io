Documentation

Everything you need to know about setting up and configuring BloxOLite products.

Introduction

MAES (Moderation and Enforcement System) is an admin framework built for Roblox developers who want full control over moderation features. It’s fast, extensible, and supports Discord ban logging.

Installation

Follow these steps to set up MAES in your game:

  1. Download MAES from the official Discord.
  2. Place the Server folder into ServerScriptService.
  3. Edit your BloxOLite module inside ServerScriptService to match your group roles and permissions.

General Configuration

return {
  ["General_Settings"] = {
    ["prefix"] = ";",
    ["Hotkey"] = "h",
    ["BanData"] = data:GetDataStore("BanData123"),
    ["Modlogs"] = data:GetDataStore("ModerationLogs123"),
    ["WebHookUrl"] = "https://discord.com/api/webhooks/...",
    ["BanPlaceEnabled"] = true,
    ["BanPlaceID"] = "1234567890",
  },
}

Rank & Permission Settings

return {
  ["Guest"] = {
    ["Rank_Needed"] = 0,
    ["Enabled"] = true,
    ["Execute_Code"] = false,
    ["Usable_Commands"] = {"panel","cmds","clear","help","credits"},
  },
}

Ranks stack — higher ranks inherit lower-rank permissions.

Security & Roblox Settings

Ensure these features are enabled:

Allow HTTPS Requests = true
Enable API Services = true

Without these, MAES won't be able to communicate with Discord.

Custom Commands

Adding a custom command

return {
  Name = "CommandName",
  Aliases = {},
  Description = "Description",
  Enabled = true,
  Function = function(player,args,Deps) return true end
}

Insert the module and add the command to a rank in BloxOLite_Config.

Anti-Cheat Docs

BloxOLite Anti-Cheat helps protect your game from exploits and receives regular updates.

Installation

  1. Download from the official Discord Server.
  2. Insert the file into your game.
  3. Drag AC_Manager into ServerScriptService and Anti-CheatClient into StarterPlayerScripts.
  4. Put the AntiCheat folder into ReplicatedStorage.

Configuration

Open ReplicatedStorage -> AntiCheat -> Config and edit the module to your preference.

ModCall Docs

ModCall allows players to request staff assistance directly from the game.

Installation

  1. Download from the official Discord Server.
  2. Insert the file into your game.
  3. Drag ModCall into ServerScriptService and ModUI into StarterGui.
  4. Put BloxOLiteRemotes into ReplicatedStorage.

Configuration

Open ServerScriptService -> ModCall -> settings and edit the module as needed.
