Introduction

BloxOLite provides a reliable moderation and anti-exploit system designed for Roblox developers. It’s built to be lightweight, secure, and easy to integrate.

Installation

1. Download the BloxOLite module from our GitHub.
2. Insert it into ServerScriptService.
3. Follow the configuration instructions below.

Commands

/kick [user] [reason]
/ban [user] [duration] [reason]
/unban [user]
/warn [user] [message]
/logs [user]

All commands are permission-based and fully configurable within the Settings module.
